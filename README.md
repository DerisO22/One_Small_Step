### CSI-370: Computer Architecture Final Project

### Technologies:
- Web Assembly
- React Three Fiber
- TypeScript
- Vite

### Goals:
1. Simulate a basic rocket launch between earth and the moon using realisitc physics and other measurements.

2. Have webassembly manage the majority of computation.

3. Have different mission phases/outcomes and provide the user with a debug menu to adjust values. 

4. Get a surface level understanding of what the engineers and scientists at NASA had to figure out when launching Saturn V.
